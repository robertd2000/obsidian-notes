Нередко возникает ситуация, когда нам надо получить данные из нескольких таблиц. Для соединения данных из разных таблиц можно использовать команду SELECT. Например, пусть имеются следующие таблицы, которые связаны между собой связями:

|   |   |
|---|---|
|1<br><br>2<br><br>3<br><br>4<br><br>5<br><br>6<br><br>7<br><br>8<br><br>9<br><br>10<br><br>11<br><br>12<br><br>13<br><br>14<br><br>15<br><br>16<br><br>17<br><br>18<br><br>19<br><br>20<br><br>21<br><br>22|`CREATE` `TABLE` `Products`<br><br>`(`<br><br>    `Id SERIAL` `PRIMARY` `KEY``,`<br><br>    `ProductName` `VARCHAR``(30)` `NOT` `NULL``,`<br><br>    `Company` `VARCHAR``(20)` `NOT` `NULL``,`<br><br>    `ProductCount` `INTEGER` `DEFAULT` `0,`<br><br>    `Price` `NUMERIC` `NOT` `NULL`<br><br>`);`<br><br>`CREATE` `TABLE` `Customers`<br><br>`(`<br><br>    `Id SERIAL` `PRIMARY` `KEY``,`<br><br>    `FirstName` `VARCHAR``(30)` `NOT` `NULL`<br><br>`);`<br><br>`CREATE` `TABLE` `Orders`<br><br>`(`<br><br>    `Id SERIAL` `PRIMARY` `KEY``,`<br><br>    `ProductId` `INTEGER` `NOT` `NULL` `REFERENCES` `Products(Id)` `ON` `DELETE` `CASCADE``,`<br><br>    `CustomerId` `INTEGER` `NOT` `NULL` `REFERENCES` `Customers(Id)` `ON` `DELETE` `CASCADE``,`<br><br>    `CreatedAt` `DATE` `NOT` `NULL``,`<br><br>    `ProductCount` `INTEGER` `DEFAULT` `1,`<br><br>    `Price` `NUMERIC` `NOT` `NULL`<br><br>`);`|

В данном случае таблицы Customers и Products связаны с таблицей Orders связью один ко многим. Таблица Orders в виде внешних ключей ProductId и CustomerId содержит ссылки на столбцы Id из соответственно таблиц Products и Customers. Также она хранит количество купленного товара (ProductCount) и и по какой цене он был куплен (Price). И кроме того, таблицы также хранит в виде столбца CreatedAt дату покупки.

Пусть эти таблицы будут содержать следующие данные:

|   |   |
|---|---|
|1<br><br>2<br><br>3<br><br>4<br><br>5<br><br>6<br><br>7<br><br>8<br><br>9<br><br>10<br><br>11<br><br>12<br><br>13<br><br>14<br><br>15<br><br>16<br><br>17<br><br>18<br><br>19<br><br>20<br><br>21<br><br>22<br><br>23<br><br>24<br><br>25<br><br>26<br><br>27<br><br>28<br><br>29<br><br>30<br><br>31<br><br>32<br><br>33<br><br>34<br><br>35|`INSERT` `INTO` `Products(ProductName, Company, ProductCount, Price)`<br><br>`VALUES` `(``'iPhone X'``,` `'Apple'``, 2, 66000),`<br><br>`(``'iPhone 8'``,` `'Apple'``, 2, 51000),`<br><br>`(``'iPhone 7'``,` `'Apple'``, 5, 42000),`<br><br>`(``'Galaxy S9'``,` `'Samsung'``, 2, 56000),`<br><br>`(``'Galaxy S8 Plus'``,` `'Samsung'``, 1, 46000),`<br><br>`(``'Nokia 9'``,` `'HDM Global'``, 2, 26000),`<br><br>`(``'Desire 12'``,` `'HTC'``, 6, 38000);`<br><br>`INSERT` `INTO` `Customers(FirstName)`<br><br>`VALUES` `(``'Tom'``), (``'Bob'``),(``'Sam'``);`<br><br>`INSERT` `INTO` `Orders(ProductId, CustomerId, CreatedAt, ProductCount, Price)`<br><br>`VALUES`<br><br>`(`<br><br>    `(``SELECT` `Id` `FROM` `Products` `WHERE` `ProductName=``'Galaxy S9'``),`<br><br>    `(``SELECT` `Id` `FROM` `Customers` `WHERE` `FirstName=``'Tom'``),`<br><br>    `'2017-07-11'``,` <br><br>    `2,`<br><br>    `(``SELECT` `Price` `FROM` `Products` `WHERE` `ProductName=``'Galaxy S9'``)`<br><br>`),`<br><br>`(`<br><br>    `(``SELECT` `Id` `FROM` `Products` `WHERE` `ProductName=``'iPhone 8'``),`<br><br>    `(``SELECT` `Id` `FROM` `Customers` `WHERE` `FirstName=``'Tom'``),`<br><br>    `'2017-07-13'``,` <br><br>    `1,`<br><br>    `(``SELECT` `Price` `FROM` `Products` `WHERE` `ProductName=``'iPhone 8'``)`<br><br>`),`<br><br>`(`<br><br>    `(``SELECT` `Id` `FROM` `Products` `WHERE` `ProductName=``'iPhone 8'``),`<br><br>    `(``SELECT` `Id` `FROM` `Customers` `WHERE` `FirstName=``'Bob'``),`<br><br>    `'2017-07-11'``,` <br><br>    `1,`<br><br>    `(``SELECT` `Price` `FROM` `Products` `WHERE` `ProductName=``'iPhone 8'``)`<br><br>`);`|

Теперь соединим две таблицы Orders и Customers:

|   |   |
|---|---|
|1|`SELECT` `*` `FROM` `Orders, Customers;`|

При такой выборке для каждая строка из таблицы Orders будет совмещаться с каждой строкой из таблицы Customers. То есть, получится перекрестное соединение. Например, в Orders три строки, а в Customers то же три строки, значит мы получим 3 * 3 = 9 строк:

![Неявное соединение таблиц без JOIN в PostgreSQL](https://metanit.com/sql/postgresql/pics/8.1.png)

То есть в данном случае мы получаем прямое (декартово) произведение двух групп. Однако вряд ли такой результат можно назвать желаемым. Тем более каждый заказ из Orders связан с конкретным покупателем из Customers, а не со всеми возможными покупателями.

Для решения этой задачи необходимо использовать выражение WHERE и фильтровать строки при условии, что поле CustomerId из Orders соответствует полю Id из Customers:

|   |   |
|---|---|
|1<br><br>2|`SELECT` `*` `FROM` `Orders, Customers`<br><br>`WHERE` `Orders.CustomerId = Customers.Id;`|

![Соединение таблиц с помощью SELECT в PostgreSQL](https://metanit.com/sql/postgresql/pics/8.2.png)

Теперь объединим данные по трем таблицам Orders, Customers и Proucts. То есть получим все заказы и добавим информацию по клиенту и связанному товару:

|   |   |
|---|---|
|1<br><br>2<br><br>3|`SELECT` `Customers.FirstName, Products.ProductName, Orders.CreatedAt`<br><br>`FROM` `Orders, Customers, Products`<br><br>`WHERE` `Orders.CustomerId = Customers.Id` `AND` `Orders.ProductId=Products.Id;`|

Так как здесь соединяются три таблицы, то необходимо применить как минимум два условия. Ключевой таблицей остается Orders, из которой извлекаются все заказы, а затем к ней подсоединяются данные по клиенту по условию `Orders.CustomerId = Customers.Id` и данные по товару по условию `Orders.ProductId=Products.Id`

![Соединение таблиц без JOIN в PostgreSQL](https://metanit.com/sql/postgresql/pics/8.3.png)

Поскольку в данном случае названия таблиц сильно увеличивают код, то мы его можем сократить за счет использования псевдонимов таблиц:

|   |   |
|---|---|
|1<br><br>2<br><br>3|`SELECT` `C.FirstName, P.ProductName, O.CreatedAt`<br><br>`FROM` `Orders` `AS` `O, Customers` `AS` `C, Products` `AS` `P`<br><br>`WHERE` `O.CustomerId = C.Id` `AND` `O.ProductId=P.Id;`|

Если необходимо при использовании псевдонима выбрать все столбцы из определенной таблицы, то можно использовать звездочку:

|                     |                                                                                                                                                                                  |
| ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1<br><br>2<br><br>3 | `SELECT` `C.FirstName, P.ProductName, O.*`<br><br>`FROM` `Orders` `AS` `O, Customers` `AS` `C, Products` `AS` `P`<br><br>`WHERE` `O.CustomerId = C.Id` `AND` `O.ProductId=P.Id;` |
