Оператор INTERSECT позволяет найти общие строки для двух выборок, то есть данный оператор выполняет операцию пересечения множеств. Для его использования применяется следующий формальный синтаксис:

|   |   |
|---|---|
|1<br><br>2|`SELECT_выражение1`<br><br>`INTERSECT` `SELECT_выражение2`|

Для примера возьмем таблицы из прошлой темы:

|   |   |
|---|---|
|1<br><br>2<br><br>3<br><br>4<br><br>5<br><br>6<br><br>7<br><br>8<br><br>9<br><br>10<br><br>11<br><br>12<br><br>13<br><br>14<br><br>15<br><br>16<br><br>17<br><br>18<br><br>19<br><br>20<br><br>21<br><br>22<br><br>23<br><br>24<br><br>25<br><br>26<br><br>27|`CREATE` `TABLE` `Customers`<br><br>`(`<br><br>    `Id SERIAL` `PRIMARY` `KEY``,`<br><br>    `FirstName` `VARCHAR``(20)` `NOT` `NULL``,`<br><br>    `LastName` `VARCHAR``(20)` `NOT` `NULL``,`<br><br>    `AccountSum` `NUMERIC` `DEFAULT` `0`<br><br>`);`<br><br>`CREATE` `TABLE` `Employees`<br><br>`(`<br><br>    `Id SERIAL` `PRIMARY` `KEY``,`<br><br>    `FirstName` `VARCHAR``(20)` `NOT` `NULL``,`<br><br>    `LastName` `VARCHAR``(20)` `NOT` `NULL`<br><br>`);`<br><br>`INSERT` `INTO` `Customers(FirstName, LastName, AccountSum)` `VALUES`<br><br>`(``'Tom'``,` `'Smith'``, 2000),`<br><br>`(``'Sam'``,` `'Brown'``, 3000),`<br><br>`(``'Paul'``,` `'Ins'``, 4200),`<br><br>`(``'Victor'``,` `'Baya'``, 2800),`<br><br>`(``'Mark'``,` `'Adams'``, 2500),`<br><br>`(``'Tim'``,` `'Cook'``, 2800);`<br><br>`INSERT` `INTO` `Employees(FirstName, LastName)` `VALUES`<br><br>`(``'Homer'``,` `'Simpson'``),`<br><br>`(``'Tom'``,` `'Smith'``),`<br><br>`(``'Mark'``,` `'Adams'``),`<br><br>`(``'Nick'``,` `'Svensson'``);`|

В таблице Customers хранятся все клиенты банка, а в таблице Employees - все его сотрудники. При этом сотрудники банка могут быть одновременно и клиентами этого банка, поэтому их данные могут храниться сразу в двух таблицах. Найдем всех сотрудников банка, которые одновременно являются его клиентами. То есть нам надо найти общие элементы двух выборок:

|   |   |
|---|---|
|1<br><br>2<br><br>3<br><br>4|`SELECT` `FirstName, LastName`<br><br>`FROM` `Employees`<br><br>`INTERSECT` `SELECT` `FirstName, LastName`<br><br>`FROM` `Customers;`|
